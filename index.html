holyChat

holyChat is a mobile application for fast communication between end users. The app is written in React Native and built with Expo. I decided to integrate Gifted Chat for all necessary chat features and added NetInfo to apply certain features while being offline.

See project on Github

meet App

A React app, designed as a PWA, for retrieving meetUps. It includes functions to limit the number of events and suggestions. The code was developed in the sense of a BDD and TDD approach (use of Jest and Cucumber). Furthermore, the app was connected to the Google Calendar API.

See project on Github See hosted project

myFlix-API

An API for retrieving film data, connected to a mongoDB database. Implementation of passport for the use of JWT tokens.

See project on Github

myFlix-Client (React)

An app hosted on Netlify, connected to the myFlix API I created, which retrieves film data from my mongoDB database. I styled it with React-Bootstrap. In the myFlix client, it is possible to retrieve film data and a profile view. Functions such as logging in, logging out, registering and de-registering users are also integrated.

See project on Github See hosted project

myFlix-Client (Angular)

A single-page, responsive movie app built with Angular, with routing and several interface views. It is designed with Angular Material and connected to the myFlix API I created, which retrieves film data from my mongoDB database. In the myFlix client, it is possible to retrieve film data and a profile view. Functions such as logging in, logging out, registering and de-registering users are also integrated.
